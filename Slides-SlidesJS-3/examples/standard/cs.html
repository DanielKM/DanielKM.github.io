Conestoga College - Online Learning

I started working on my computer science education because throughout my science career I have always excelled when working with technology. I wanted to open up opportunities to improve myself while staying on the cutting edge of science and tech. Learning coding has opened up options I would not have had if I had stayed as a pure scientist. I am able to work on personal projects, help out friends and colleagues, and am now looking at switching careers into the tech field permanently. I have taken two courses through Conestoga College to prepare me for a career in web development.

Advanced CSS3 and HTML5

This course builds upon an intermediate knowledge of HTML and CSS to create basic mobile pages and applications. It covers the use of native video, forms, calendars, and more. CSS3 features are taught to allow fast-loading, exciting, interactive websites, as well as mobile sites. The course also covers the use of jQuery mobile to create app-like mobile web pages and themes.

JavaScript

This course provides the knowledge necessary to design and develop dynamic websites using client-side JavaScript. It introduces students to the fundamentals of JavaScript programming including client-side JavaScript programming model, language syntax, variables, functions, control flow statements and build-in objects. The course provides hands-on experience writing client-side JavaScript to create special effects, personalize web pages and design intelligent web-forms.

Wilfrid Laurier University

Biostatistics

This course examined the totality of issues related to data collection, management and analysis for biologists: from the development of hypotheses, the design of experiments, measurement and sampling, to statistical analyses and final publication of results in formats acceptable for scientific reports and papers. The statistical analyses were performed using the R programming language.
